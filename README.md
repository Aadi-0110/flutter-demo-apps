# Flutter Demo Apps

A collection of Flutter demo applications showcasing various features and implementations.

## 📱 Projects

### Flutter Notes App
A feature-rich notes application built with Flutter.

**Features:**
- Create, read, update, and delete notes
- Clean and intuitive user interface
- Local data persistence
- Material Design 3 components

**Getting Started:**

```bash
cd flutter-notes-app
flutter pub get
flutter run
```

## 🛠️ Prerequisites

- Flutter SDK (latest version recommended)
- Dart SDK
- Android Studio or VS Code with Flutter extensions

## 📋 Requirements

- Flutter: >=3.0.0
- Dart: >=3.0.0
